Summary of Water Usage From 2006 to 2011

  • USA: data at the county level are compiled into a national water-use data system every five years. Year 2006 - 2008 are set the same as Year 2005. Year 2009 - 2011 are set the same as Year 2010.
  • Canada: Data are collected every two years. Year 2006 is set the same as Year 2007. Year 2008 is set the same as Year 2009. Year 2010 is set the same as Year 2011.
  • Germany: Data are collected every three years. Year 2006 and Year 2008 are set the same as Year 2007. Year 2009 and Year 2011 are set the same as Year 2010.
